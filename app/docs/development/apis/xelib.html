xelib

xelib is a JavaScript wrapper around XEditLib.dll, a DLL which exposes an API to use the XEdit framework.

Namespaces

All members are top-level in xelib. This choice was made to simplify and shorten code which calls xelib functions.

Important Concepts

Elements

With xelib and the XEdit framework, everything in plugin files (including the files thesmelves) is an element. In zEdit's GUI, every row in the tree view and every row in the record view is an element. Elements can be files, groups, records, structs (a collection of a specific set of elements), arrays (a collection of some number of a single type of element), or values. Elements always have certain things, such as a name and a value.
