Development

zEdit is a NodeJS Electron application. zEdit is only built for x86 Windows due to XEditLib targetting x86 Windows.

Technologies

zEdit uses the following technologies:

  • ES6 JavaScript - Programming Language
  • NodeJS - JavaScript Runtime
  • Electron - Platform
  • AngularJS 1.5 - JavaScript Framework
  • SASS - CSS Extension Language
  • CSS3 - Stylesheet Language
  • HTML5 - Markup Language

Contributing

There are several ways developers can contribute to zEdit:

  • Writing Scripts
  • Creating Modules
  • Contributing to the main codebase
