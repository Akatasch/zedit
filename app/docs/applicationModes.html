zEdit will soon offer a variety of application modes other than the currently included "Edit" mode. Application modes will allow zEdit to offer smooth and complete user experiences for use cases other than viewing/editing plugin files.

All application modes will be accessible from the Start View. zEdit will likely be renamed once it includes more application modes than just "Edit". The currently planned name is "zTools".

Currently planned/implemented application modes include:

Application Mode Description
Edit This application mode offers functionality for viewing, editing, and saving plugin files. See Edit View for more information.
[Planned] Clean Planned application mode which will offer plugin cleaning functionality. Will be merged from Shampoo Plugin Cleaner.
[Planned] Merge Planned application mode which will offer functionality similar to Merge Plugins Standalone.
[Planned] Smash Planned application mode which will offer functionality similar to Mator Smash.
